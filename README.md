# Employee Management System

## Overview
The Employee Management System is a web-based application that allows users to manage employee records efficiently. The system provides functionalities for adding, updating, and deleting employee details.

## Features
- Add, view, update, and delete employee records.
- Display the total count of employees.
- User authentication system.
- **Admin Panel**: Admins can manage all employee details.

## Technologies Used
- **Frontend**: JSP, HTML, CSS
- **Backend**: Java (Servlets, JSP)
- **Database**: MySQL
- **Server**: Apache Tomcat
- **ORM**: Hibernate

## Usage
- **Login as Admin**: Access the admin panel where you can add, update, or delete employees.
- **Manage Employees**: View the list of employees and perform CRUD operations.
- **Logout**: Securely log out after managing the system.
